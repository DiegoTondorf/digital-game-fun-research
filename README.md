# Digital Game Fun Research

This repository contains replication packages for academic research focused on **fun in digital games**. The studies hosted here aim to support transparency, reproducibility, and the long-term organization of related research works.

## About This Repository

Each study is organized in a dedicated, self-contained folder. This structure allows researchers to independently navigate, reproduce, and build upon individual works without requiring knowledge of the full repository.

> **Note on versioning:** This GitHub repository is the **mutable** version of the research artifacts and may be updated over time (e.g., to fix errors or add supplementary material). Each study also has an **immutable archived version** on [Zenodo](https://zenodo.org), which is assigned a permanent DOI and is suitable for citation in academic publications.

## Repository Navigation

- Each top-level folder corresponds to a separate study.
- Inside each study folder you will find a dedicated `README.md` with full details on the study, reproduction instructions, and a description of its contents.
- Raw data, analysis scripts, and output results are separated into distinct sub-folders for clarity.

## Reproducibility Statement

All replication packages in this repository are made publicly available to encourage independent verification of results and reuse in future research. Researchers are welcome to reproduce the analyses, extend the datasets, and build upon the methods described herein. If you use any of these materials, please cite the corresponding published study and its archived Zenodo version.
